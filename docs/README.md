# Documentação do Projeto

Esta pasta contém toda a documentação técnica e instruções do projeto Bootstrap.

## Estrutura

- **INSTRUCOES_GITHUB_PAGES.md** - Instruções para configuração e deploy no GitHub Pages
- **specs/** - Especificações técnicas do projeto (requirements, design, tasks)

## Organização

Toda documentação técnica, guias e instruções devem ser mantidos nesta pasta para manter a raiz do projeto limpa e focada nos arquivos da aplicação web.

## Arquivos na Raiz

A raiz do projeto deve conter apenas:
- Arquivos HTML (login.html, index.html)
- Pastas de recursos (css/, js/, assets/)
- Arquivos de configuração essenciais (.gitignore, README.md principal)
