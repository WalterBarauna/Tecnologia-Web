Gerador de Favicon

Clique no botão para gerar e baixar o favicon.ico

Instruções:

  1. Clique no botão "Gerar e Baixar Favicon"
  2. O arquivo favicon.ico será baixado automaticamente
  3. Mova o arquivo para a pasta assets/imagens/
  4. Substitua o arquivo existente se houver

Nota: O favicon gerado é uma letra "W" branca em fundo azul (#0d6efd)
