# Produto

## Visão Geral

Portfólio web pessoal e currículo online desenvolvido como projeto acadêmico para a disciplina de Tecnologia Web. O sistema consiste em duas páginas principais:

1. **Página de Login** (`index.html`): Interface de autenticação simples sem backend real
2. **Landing Page** (`portfolio.html`): Currículo profissional online com seções de apresentação, formação, habilidades, experiência e contato

## Objetivo

Demonstrar competências em desenvolvimento front-end, design responsivo e deploy de aplicações web estáticas utilizando Bootstrap 5.

## Público-Alvo

- Recrutadores e empregadores
- Professores e avaliadores acadêmicos
- Colegas e profissionais da área de tecnologia

## Características Principais

- Interface responsiva que se adapta a mobile, tablet e desktop
- Design moderno com gradientes e animações suaves
- Navegação intuitiva com scroll suave entre seções
- Links para redes sociais (LinkedIn, Instagram, Email)
- Sistema de login com feedback visual (sem autenticação real)
- Otimizado para performance e SEO

## Fluxo de Uso

```
Usuário acessa index.html → Preenche formulário de login → 
Clica em "Entrar" → Redirecionado para portfolio.html → 
Navega pelas seções do currículo → Acessa links de contato
```

## Hospedagem

Publicado no GitHub Pages com deploy automático via branch main.
